Flex Documentation

Flex documentation, including Release Notes, is available from the Flex Help and Support page and the Flex Developer Center.

Using Adobe Flex 4.5 Describes application development basics, such as MXML and ActionScript syntax, creating applications, using Flex components, the Flex data model, error handling, and Flex Advanced Data Visualization Components. Also contains information on custom component creation, data visualization tools, Flex charting components, automation, and application deployment.
Accessing Data with Adobe Flex Provides information on using Flash Builder data wizards to create data-centric Flex applications. Also includes information on accessing data using RPC services and web services.
Using Flash Bulider 4.5 Provides information on using Adobe Flash Builder 4.5 to create Flex applications.

Developing Mobile Applications with Flex

Provides information on using Flex to build standalone mobile applications running on Adobe AIR. Additionally, provides information on using Adobe Flash Builder 4.5 to develop, debug and deploy mobile applications.

Tutorials on data access with Flash Builder

Provides tutorials and samples on accessing data with Flash Builder 4.5 including how to define data services and manage accessing data.
ActionScript 3.0 Reference for the Adobe Flash Platform Describes the MXML and ActionScript 3.0 programming interfaces, including the API reference for core ActionScript language elements (HTML-only).
